Requirements for Linux/FreeBSD/OSX
==================================

* GCC 4.6 or newer (Clang can be used instead of GCC)
* Boost 1.46 or newer
* crypto++

Requirements for Windows
========================

* VS2013 (known to work with 12.0.21005.1 or newer)
* Boost 1.46 or newer
* crypto++ 5.62
